# airtoday
Platform to measure measure the noise, fine dust, and toxic gas level focusing on the local area.  
Utilized Stacks: Embedded System, Docker, Kubernetes, TensorFlow, React, AWS EC2, IoT Core, Kibana, and LoRa.
